## Features

- Light/dark mode toggle
- Responsive layout
- Accessible design
- Made using CSS3, HTML5, and JavaScript and deployed on Netlify
- Contact form with spam honeypot field

## Installation

Clone this repo.

```bash
git clone https://github.com/josh-mock/portfolio.git
cd portfolio
```
